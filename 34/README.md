   <details><summary>Пример</summary>

   ```java
   int x = 0;
   int speed = 2;

   void setup() {
     size(480, 200);
   }

   void draw() {
     background(255);

     x = x + speed;

     if (x < 0) {
       x = 0;
     } else if (x > width) {
       x = width;
     }

     stroke(0);
     fill(175);
     ellipse(x, 100, 32, 32);
   }
   ```

   </details>
